Design

Overview

The Burnboard was designed from the ground up with a few key features in mind. First, the heart of the Burnboard is a WCH32V203G6U6 processor. This processor is an affordable processor operating under the RISC-V instruction set. RISC-V is an open ISA that is well-documented and increasingly supported in a variety of applications. In an attempt to further open-source efforts, this processor was selected.

Another core tenent of the Burnboard's design is the use of embedded Rust for the custom firmware. The project exists as a sort of 'full-stack' work, from the hardware to the user design. The existence of the ch32-hal crate greatly facilitated this decision, as overloading the difficulty of this project with such in-depth programming before ever realizing the functionality of the keyboard would've been a major mistake. Taking advantage of this crate allows for simple asynchronous functionality to be implemented. This functionality lends itself well to the third major tenent of this project: wireless connectivity. The CH32V203G6U6 is not a bluetooth enabled chip, so the BM71 bluetooth module was chosen to enable bluetooth functionality of the device. Furthermore, a system that handles a battery is a necessary feature of the project. The BM71 can communicate via the Universal Asynchronous Receiver/Transmitter, which the CH32V203 is capable of.
